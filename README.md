# Flutter学习demo

## 当前功能

1. 简单的App框架，包括首页、用户中心、登陆注册页等     [x]
2. 调起手机摄像头拍照、拍视频，选择图片、视频上传      [x]
3. 启用手机蓝牙功能，实现蓝牙设备连接与数据传输        [ ]
4. 手机NFC的读取与写入                           [ ]
5. RTMP直播流播放                               [ ]
6. 地图、定位                                   [ ]
7. 其他传感器（陀螺仪、加速器等）                   [ ]
8. 自更新                                      [ ]
9. 识别二维码、条形码                            [ ]
10. 生成二维码、条形码，并保存到设备中              [ ]
